# vs-theme-eal-magazine

Vivliostyle用テーマ `vs-theme-eal-magazine` は、複数の段組と浮動要素を配置を多用した、雑誌のような複雑なレイアウトに適したテーマです。

このテーマはEAL（EPUB Adaptive Layout）を使用したテーマのため、EALの仕様に沿ってCSSを記述する必要があります。
また、このテーマは[vs-theme-eal-float-system](https://github.com/spring-raining/vivliostyle-themes/tree/main/themes/eal-float-system)をベースとして作成されています。
vs-theme-grid-systemについては、「[EPUB Adaptive Layoutを使った雑誌風レイアウト](./eal-float-system)」も参考にしてください。

### インストール方法

`vivliostyle.config.js` に以下のように設定します。

```js
module.exports = {
  theme: '@spring-raining/vs-theme-eal-magazine@1',
};
```

デフォルトでは横書き用の設定ファイルが読み込まれます。縦書き用の設定ファイルを読み込むには以下のように設定します。

```js
module.exports = {
  theme: {
    specifier: '@spring-raining/vs-theme-eal-magazine@1',
    import: 'theme-v.css',
  },
};
```

### 使用方法

デフォルトでは、カラム幅が `12rem` になるように自動的にカラム数が設定されています。カラム幅やカラム数を変更する場合は、以下のようにCSSとEAL両方の設定を変更します。

```css
:root {
  --vs-columns: 3;
  --vs-column-gap: 10mm;
}
@-epubx-define {
  vs-columns: 3;
  vs-column-gap: 10mm;
}
```

利便性のため、テーマにはいくつかのユーティリティクラスが用意されています。

* `.tcy`: 縦書き文章における縦中横を設定（`text-combine-upright: all`）
* `.upright`: 縦書き文章で、横書き用の文字を縦書き用の字形と同様に正立で配置する（`text-orientation: upright`）
* `.drop-cap`: 設定した要素のドロップキャップ（文章の1文字目を大きくするスタイル）を有効化する
* `.lead`: リード文用のスタイルを適用する
* `.floating-quote`: 引用を効果的に表示するスタイルを適用する
* `.pull-inline-start`/`.pull-inline-end`: 要素を左（右）（縦書きでは上（下））に少しはみ出したようなfloatを設定する
* `.break-before-page-h`/`.break-before-page-v`: 横書き（縦書き）文章で強制的に改ページを設定する

Float Systemに関するユーティリティクラスは、[EPUB Adaptive Layoutを使った雑誌風レイアウト](./eal-float-system)で紹介されているものが使用できます。
